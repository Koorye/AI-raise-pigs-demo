本项目仅供学习使用，严禁商用，制作本项目的初衷仅为完成软件架构设计的小作业。

本项目的一切功能均未实现，仅为原型。

本项目使用基于 Vue2 的 Uni-app 开发，使用如下插件：

- `秋云 ucharts` 高性能跨全端图表；
- `uni-easyinput` 输入框；
- `uni-list` 列表；
- `uni-search-bar` 搜索框；
- `uni-datetime-picker` 日期选择器。

本项目所有图标来自 `Iconfont` 阿里巴巴矢量图标库。
